🇱🇹 Drone Defense: Povilas & Romas

Two legends.
One sky full of drones.
Zero chill.

Welcome to the most elite (and slightly chaotic) Lithuanian defense system ever built.

🎮 The Game

Povilas (tall & composed)
Romas (short, bold & unstoppable)

Together they defend Lithuanian cities from incoming drone swarms.

Starting at Gediminas Castle in Vilnius, you’ll fight your way across Lithuania:

🏰 Vilnius
🌉 Kaunas
⚓ Klaipėda
☀️ Šiauliai

Each level gets harder.
More drones. Faster drones. Less mercy.

🕹️ Controls
Player 1 — Povilas
Move: ← →
Shoot: SPACE
Player 2 — Romas
Move: A / D
Shoot: R

💥 Rules
Each player has 3 lives
Shoot drones before they hit the ground
Destroy enough drones → unlock next city
Game ends when both players are out

🚀 Run the Game
pip install pygame
python main.py
