	🇱🇹 Drone Defense: Povilas & Romas
Two legends.
One sky full of drones.
Zero chill.<br/>

Welcome to the most elite (and slightly chaotic) Lithuanian defense system ever built.

	🎮 The Game

Povilas (tall & composed)
Romas (short, bold & unstoppable)
Together they defend Lithuanian cities from incoming drone swarms.
Starting at Gediminas Castle in Vilnius, you’ll fight your way across Lithuania:<br/>

🏰 Vilnius<br/>
🌉 Kaunas<br/>
⚓ Klaipėda<br/>
☀️ Šiauliai<br/>

Each level gets harder.
More drones. Faster drones. Less mercy.

	🕹️ Controls
Player 1 — Povilas<br/>
Move: ← →<br/>
Shoot: SPACE<br/>
Player 2 — Romas<br/>
Move: A / D<br/>
Shoot: R<br/>

	💥 Rules
Each player has 3 lives<br/>
Shoot drones before they hit the ground<br/>
Destroy enough drones → unlock next city<br/>
Game ends when both players are out<br/>

	🚀 Run the Game
pip install pygame<br/>
python main.py<br/>
